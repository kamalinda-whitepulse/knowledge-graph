# Personal Knowledge Graph Builder

## Overview

Personal Knowledge Graph Builder is a web-based knowledge management application that enables users to create notes, establish relationships between notes, and visualize knowledge as an interconnected graph.

The system helps users organize information more effectively by transforming isolated notes into a network of connected concepts.

---

## Features

### Authentication

* User Registration
* User Login
* JWT Authentication
* Protected Routes

### Notes Management

* Create Notes
* View Notes
* Update Notes
* Delete Notes
* Add Tags

### Relationship Management

* Create Relationships Between Notes
* Remove Relationships
* Support Multiple Relationship Types:

  * Related To
  * Depends On
  * References
  * Parent Of

### Search

* Search by Title
* Search by Content
* Search by Tags

### Graph Visualization

* Interactive Knowledge Graph
* Visual Representation of Note Connections
* Node and Edge Navigation

### Dashboard

* Total Notes
* Total Relationships
* Most Connected Notes
* Recently Created Notes
